## Título

Audio con Strudel: live coding sonoro y estructuras temporales.

## Momento del curso

Investigación.

## Propósito

Explorar Strudel como entorno de creación sonora en tiempo real y comprender el
audio como una fuente de organización temporal para un sistema performático. En
esta unidad el énfasis estará en patrones, eventos, ciclos, capas, variación y
control en vivo.

## Tiempo

2 semanas.

## Evaluación

Esta unidad aporta el **12.5%** de la nota final.

| Componente | Valor | Evidencia |
|---|---:|---|
| Conceptual | 6.25% | Examen sobre conceptos de live coding, patrones, eventos temporales, ciclos y posibilidades performáticas de Strudel. |
| Aplicación | 6.25% | Mini estudio sonoro en Strudel, documentado en la bitácora de GitHub. |

decisiones sonoras, los errores encontrados y las evidencias del resultado.
</Aside>

### Actividad 03

En esta unidad vamos a realizar experimentos con audio en Strudel. Te voy a mostrar algunos ejemplos:

```javascript
setcpm(35) // 140 beat

// ==========================================
// MEZCLADORA INDEPENDIENTE
// ==========================================
const BASS_ON    = 0
const PIANO_ON   = 0
const CEL = 0
const VAR3 = 0
const VAR1  = 0
const VAR2 = 0

// Bass: Sigue las tónicas de la progresión (Fmaj7 - G6 - Em7 - Am9)
bass: note("{<f1 g1 e1 a1>}")
  .sound("supersaw")
  .lpf(slider(1187.4,300,2000)) 
  .room(1)
  .postgain(BASS_ON ? 0.3 : 0)

piano : note("{<[f3,a3,c4,e4] [g3,b3,d4,e4] [e3,g3,b3,d4] [a3,c4,e4,b4]>}")  
  .sound("piano")
  .attack(slider(0.192,0,1)) 
  .lpf(slider(1490,300,2000))
  .room(1.2) 
  .postgain(PIANO_ON ? 0.5 : 0)

// ==========================================
// CÉLULAS Y VARIACIONES MELÓDICAS
// ==========================================
const celula = "<[c5 e5] [b4 g5] [a4 d5] [b4 e5 c5]>"
const var1   = "< [[c5 e5] [b4 g5]] [[c5 e5] [b4 g5]] [[a4 d5] [b4 e5 c5]] [[a4 d5] [b4 e5 c5]]>"
const var2   = celula.transpose(12)
const var3   = "{<[e5 c5] [f5 a4] [g5 d5] [f5 c5 e5]>}"

// ==========================================
// CANALES DE MELODÍA INDEPENDIENTES
// ==========================================

// Célula
$celula: note(celula)
  .sound("sine") 
  .decay(0.4)
  .sustain(0.2)
  .room(1.5) 
  .postgain(CEL ? 0.6 : 0)

// Disminución rítmica
$var1: note(var1)
  .sound("sine") 
  .decay(0.4)
  .sustain(0.2)
  .room(1.5) 
  .postgain(VAR1 ? 0.5 : 0)

// Transpuesta (Octava Alta)
$var2: note(var2)
  .sound("triangle") // sine, triangle
  .decay(0.4)
  .sustain(0.2)
  .room(1.5) 
  .postgain(VAR2 ? 0.2 : 0) 

// Inversión
$var3: note(var3)
  .sound("sine") 
  .attack(slider(0.359,0,1))
  .lpf(slider(993.6,300,2000))
  .postgain(VAR3 ? 0.3 : 0)
```

  s("bd")
    .beat("0,7,10?,15",16)
    .gain("<1 .95 1 .9>")

  s("bd")
    .beat("14?",16)

  s("hh")

  s("cp")
    .beat("4,12",16)
    .gain("<.75 .7 .82 .72>"),

  // Clap fantasma antes de la caja
  s("cp")
    .beat("3?,11?",16)
    .gain(.16)
    .late(.012)

.bank("RolandTr909")

### Actividad 04

En esta esta actividad vas a comenzar a explorar algunas ideas de creación sonora. Vamos a revisar algunas cuestiones 
juntos, pero realmente la invitación es a que juegues y experimentes. 

Ingresemos a este sitio web:

<LinkPreview 
id="https://learningmusic.ableton.com/" 
title="Get started making music" 
/>

Vamos a reproducir los mismos beats usando esta base:

```javascript
setcpm(24.5)

let drum = stack(
  s("oh:1").beat("3,5,9,13",16),
  s("hh").beat("0,1,4,6,8,10,12,14",16),
  s("cp").beat("4,12",16),
  s("bd").beat("0,2,3,7,9,10,15",16),
).bank("RolandTr909");


$drum: drum
```

Luego puedes ir agregando capas mediante notas, acordes, variaciones y efectos. Te 
dejo algunos snippets de ejemplo para que los uses como referencia:

```javascript
setcpm(20)
let melody = note("[c4 c4 e4 g4  e4 e4 ~ d4 c4 a3 a3 a3 ~ d4 d4 c4]").sound("piano").legato(1);
$melody: melody
```
